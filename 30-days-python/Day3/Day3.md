# Day 3 Challenge: Character Counter

## Overview
Your task for Day 3 is to build a **Character Counter**. This program should read a text file and calculate the total number of characters it contains. This exercise helps you understand how text is represented at the most granular level.

---

## Challenge Requirements
- Create a text file (`sample.txt`) with some content.  
- Write a Python script (`char_counter.py`) that:  
  - Opens the file.  
  - Counts the total number of characters (including spaces and punctuation).  
  - Prints the result.  

---

## Hints
- Decide whether to read the file all at once or line by line.  
- Remember that Python has built‑in ways to measure string length.  
- Consider whether you want to count spaces and punctuation or exclude them — document your choice.  

---

## Goal
By the end of this challenge, you should be comfortable with:
- Reading text files.  
- Measuring string lengths.  
- Understanding the difference between characters, words, and lines.  

---

👉 This challenge builds on Day 1 (words) and Day 2 (lines), giving you a complete view of text analysis at different levels.
