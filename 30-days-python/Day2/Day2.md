# Day 2 Challenge: File Line Counter

## Overview
Your task for Day 2 is to build a **File Line Counter**. This program should read a text file and determine how many lines it contains. It’s a simple but essential exercise that strengthens your understanding of file handling and iteration in Python.

---

## Challenge Requirements
- Create a text file (`sample.txt`) with multiple lines of text.  
- Write a Python script (`line_counter.py`) that:  
  - Opens the file.  
  - Counts the number of lines.  
  - Prints the total line count.  

---

## Hints
- Think about how to open a file safely.  
- Consider whether you want to read the file all at once or line by line.  
- Remember that Python provides ways to count items in a list.  

---

## Goal
By the end of this challenge, you should be comfortable with:
- Reading files line by line.  
- Using loops or built‑in functions to count.  
- Printing results in a clear format.  

---

👉 This challenge builds directly on Day 1, moving from word counting to line counting — a step toward log processing and text analytics.
